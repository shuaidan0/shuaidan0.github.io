Setup your website using 'clean white' theme

setup your website

Posted by     "" on Sunday, October 16, 2022

Install

brew install hugo  
hugo version  
hugo new site myhugowebsite

Download theme

cd myhugowebsite  
git init  
cd themes  
git clone https://github.com/hopeworker/hugo-theme-cleanwhite.git  
cp -r hugo-theme-cleanwhite/exampleSite/** ../

Add post

hugo new post/my-first-post.md

Launch

hugo server -D

Build static pages

hugo -D

Reference

https://gohugo.io/getting-started/quick-start/ https://themes.gohugo.io/ https://themes.gohugo.io/themes/hugo-theme-cleanwhite/ https://themes.gohugo.io/themes/hugo-theme-cleanwhite/#comment-systems
